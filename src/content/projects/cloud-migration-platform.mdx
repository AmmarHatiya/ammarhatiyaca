## Overview

Designed and implemented a comprehensive cloud migration platform that automated the assessment, planning, and execution of workload migrations from on-premises VMware environments to AWS.

## Technical Approach

- Built a Python-based discovery agent that inventoried 200+ VMs, mapping dependencies and network topology automatically
- Created Terraform modules for repeatable infrastructure provisioning across multiple AWS regions
- Implemented a blue-green migration strategy using AWS Application Migration Service with custom pre/post-migration validation scripts
- Designed CloudFormation templates for consistent networking (VPC, Transit Gateway, Direct Connect) across 4 AWS accounts

## Results

- **Zero-downtime migration** for all production workloads over a 6-month period
- **$18k/month savings** through right-sizing, Reserved Instances, and eliminating legacy hardware maintenance
- **40% faster provisioning** compared to manual infrastructure setup
- Migration runbooks and IaC templates now used as the standard for all new cloud deployments
